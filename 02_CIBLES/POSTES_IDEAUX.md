# POSTES_IDEAUX

## Profils de poste ciblés

- **Director / Head of Last Mile Operations**
  - Niveau : Director / Senior Manager / Head of
  - Responsabilités principales : Lancement et opérations réseau last-mile, stations de livraison, DSP management, performance opérationnelle
  - Compétences clés : Last-mile delivery, delivery station, sortation, dispatch, French labour law, cost optimisation
  - Critères de réussite : Réseau opérationnel, KPIs livraison, réduction coûts

- **Head of AMZL / Delivery Network Expansion**
  - Niveau : Senior Manager / Director
  - Responsabilités principales : Déploiement de nouveaux sites, coordination cross-fonctionnelle, playbooks, compliance
  - Compétences clés : Launch management, Zero Defect Start Up, network planning, capacity planning
  - Critères de réussite : Stations livrées on-time, peak season readiness

- **Portfolio Manager / Senior Portfolio Manager Transformation**
  - Niveau : Portfolio Manager / Senior Portfolio Manager
  - Responsabilités principales : Gouvernance portfolio, stage gates, SteerCo, benefits realisation, financial management
  - Compétences clés : Portfolio governance, RAID, CapEx/OpEx, regulated environment, PMO
  - Critères de réussite : Score ATS ≥ 7, delivery on-time, budget maîtrisé

- **Program Director / Global Program Director**
  - Niveau : Director / Senior Manager
  - Responsabilités principales : Pilotage de programmes de transformation complexes, multi-pays, cross-fonctionnel
  - Compétences clés : Programme management, stakeholder management, change management, executive reporting
  - Critères de réussite : Programmes livrés, adoption par les parties prenantes

- **Director Business Excellence / Operational Excellence**
  - Niveau : Director
  - Responsabilités principales : Excellence opérationnelle, transformation, BI, automatisation, performance client
  - Compétences clés : Continuous improvement, automation, BI, KPI, data-driven decision making
  - Critères de réussite : Réduction coûts mesurable, amélioration performance opérationnelle

- **Head of Supply Chain Transformation**
  - Niveau : Director / Head of
  - Responsabilités principales : Transformation supply chain, digitalisation, intégration systèmes, change management
  - Compétences clés : Supply chain, transformation, WMS/TMS, ERP, project management
  - Critères de réussite : Transformation déployée, adoption utilisateurs, ROI mesuré

- **Director of Expansion / Network Development**
  - Niveau : Director / Senior Manager
  - Responsabilités principales : Expansion réseau logistique, CAPEX, multi-pays, coordination équipes cross-fonctionnelles
  - Compétences clés : CAPEX management, expansion, real estate, construction coordination, multi-country
  - Critères de réussite : Sites livrés dans les délais et budgets

- **PMO Director / Head of PMO**
  - Niveau : Director / Head of
  - Responsabilités principales : Gouvernance PMO, reporting exécutif, priorisation portfolio, outils de gestion
  - Compétences clés : PMO, portfolio governance, reporting, tooling, stakeholder management
  - Critères de réussite : Visibilité portfolio, qualité des décisions, adoption des outils
